# Testing

This document provides a regression testing checklist for COSMIC Files. The checklist provides a starting point for Quality Assurance reviews.

## Checklist

### Window Management

#### Tiling

- [ ] Super+Y enables/disables tiling.

#### Swapping

- [ ] In a tiled workspace, Super+X allows swapping windows.
- [ ] Super+X allows swapping windows between two different tiled workspaces.
- [ ] Open two windows in a tiled workspace. Press Super+U to select both at once. Press and hold Super+X, then use the Down arrow key to move them to a new workspace.
- [ ] Do the same thing again, but before releasing Super+X, move them back up to the previous workspace.
